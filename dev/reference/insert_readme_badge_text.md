# Insert text to badge section of README lines

Insert text to badge section of README lines

## Usage

``` r
insert_readme_badge_text(badge_text = NULL, readme_lines = NULL, path)
```

## Arguments

- badge_text:

  A character value for text to insert into badge section of README
  lines.

- readme_lines:

  A character vector of lines of text in README.

- path:

  Path to file to add repostatus badge to.

## Value

A character vector of README lines of text.

## Examples

``` r
if (FALSE) get_readme_badge_lines(readme_lines)
```
