# Get file path to README file

Get file path to README file

## Usage

``` r
get_readme_path(path = NULL)
```

## Arguments

- path:

  Path to README file. Set to NULL by default which would indicate that
  a README file is in the root directory of the project.

## Value

Path to README file.

## Examples

``` r
if (FALSE) get_readme_path()
```
