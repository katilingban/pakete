# Save Codeberg mirror GitHub Actions workflow

Save Codeberg mirror GitHub Actions workflow

## Usage

``` r
save_gha_workflow_codeberg_mirror(gha_text)
```

## Arguments

- gha_text:

  Codeberg mirror GitHub Actions workflow text to save into
  mirror-codeberg.yaml.

## Value

A mirror-codeberg.yaml file in the `.github/workflows` directory

## Examples

``` r
if (FALSE) save_gha_workflow_codeberg_mirror(gha_text)
```
